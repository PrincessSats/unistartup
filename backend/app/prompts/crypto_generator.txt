Generate a challenge as a single valid JSON object. Do NOT include any text outside the JSON.

{
  "title": "<short challenge title, 3-8 words>",

  "crypto_chain": [

Rules:

5. Do NOT include the flag or ciphertext in the description

Respond with ONLY the JSON object, no markdown fences, no explanation.
